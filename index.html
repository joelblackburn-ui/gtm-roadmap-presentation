3. Data Scoring Model

Planning Three-layer scoring to reflect true likelihood of success.

Layer 1: ICP Fit

Firmographic match to ideal customer. ATS used, avg hires/year, new Talent Director, funding, security breach.

Layer 2: Awareness

Marketing engagement at account level. Content downloads, webinars, email engagement, ad interactions.

Layer 3: Intent

Active buying signals. Website visits, competitor connections, Gong pain confirmation, contract renewals.

Combined Score = ICP Fit + Awareness + Intent

High ICP + High Awareness + High Intent = Top priority
High ICP + Low Awareness = Nurture candidate

Prerequisites

  • Source of Truth must be reliable
  • Enrichment uplift on firmographics
  • Data point collection infrastructure

Success Metrics

  • Score-to-conversion correlation
  • Top-scored → higher meeting rate
  • More meetings from same effort

4. Optimised Execution

Planning Fix fundamental limitations in current tooling.

Current State (Outreach Only)

  • LinkedIn not tracked — Conversations lost
  • No email validation — Bounces damage reputation
  • Single inbox/user — Volume capped
  • Shared domain — One bad actor hurts everyone
  • Tab-switching — Inefficient reply management

New Stack

  • HeyReach — LinkedIn with tracking
  • Instantly — Cold email, domain isolation
  • OutboundSync — HubSpot integration
  • Unified Inbox — All replies in one place

Key benefit: Each user's reputation protected. Bad practices don't damage the team.

5. Automated Routing & Sequencing

Capstone Where it all comes together.

Prerequisites (Why This Is Last)

  • Source of Truth — Can't route to garbage data
  • Scoring Model — Need accurate scores to prioritise
  • Optimised Execution — Need infrastructure to run sequences
  • SDR Workflow — Need efficient human touchpoints

The Plan

  • Weekly cron job — Best 200 accounts/SDR
  • High score + intent → Priority human outreach
  • High score + low engagement → Automated nurture
  • Low score → Light-touch sequence

Human-in-the-Loop

SDRs can override allocations. Manual sequence selection always available. Outcomes feed back into the scoring model. The system learns.
